# 暗記君 ONLINE

### 前作,暗記君は[コチラ](http://ux.getuploader.com/ankikun/download/1/%E6%9A%97%E8%A8%98%E5%90%9B.rar)からダウンロード可能です。

## 環境
本アプリケーションは、  
Mac OS X 10.7.4,  
Ruby 1.9.3-p194,  
Rails 3.2.8  
で動作を確認しています。

## 使い方
1. $ git clone git@github.com:217/ankikun.git
2. $ cd ankikun
3. $ bundle install
4. $ rails s
5. ブラウザをオープン
6. http://localhost:3000/ にアクセス！
